# Unlighter

## Development Setup

### Compiles and hot-reloads for development

First, clone the project :

```bash
git clone https://github.com/Dono7/Unlighter.git
cd unlighter
```

Then with Yarn (recommanded) : 

```bash
yarn install
yarn electron:serve
```

Or with NPM : 

```bash
npm install
npm run electron:serve
```

### Compiles and minifies for production

```bash
npm run build
# or
yarn build
```
